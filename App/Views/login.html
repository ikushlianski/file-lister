What I used

What I used to style the app

I tried to make this app look nice using the following technologies:

  • Twig for server-side templating
  • Responsive design, including media and feature queries
  • Materialize CSS framework to make the app look really nice (never worked with it before, so it was a chance to learn it hands-on)
  • Material icons
  • Fixed action button (FAB), to improve UI on mobile devices
  • Materialize's 'toasts' instead of dull static messages for users
  • Cute login form with an awesome on and off button for long-term auth
  • SASS (with sourcemaps) for quicker styling
  • Gulp task runner to collect, concat and minify js and css

The app works fine in modern browsers (even in IE11) and in major mobile browsers.
